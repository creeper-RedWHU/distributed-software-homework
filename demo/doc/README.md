```
                         ┌─────────────┐
                         │   Client    │
                         └──────┬──────┘
                                │
                         ┌──────▼──────┐
                         │    Nginx    │ 负载均衡 + 动静分离
                         └──┬──────┬───┘
                            │      │
                    ┌───────▼┐  ┌──▼───────┐
                    │  App1  │  │   App2   │  Spring Boot 多实例
                    └───┬────┘  └────┬─────┘
                        └──────┬─────┘
                               │
         ┌────────┬────────────┼────────────┬──────────┐
         ▼        ▼            ▼            ▼          ▼
    ┌─────────┐┌──────┐┌────────────┐┌──────────┐┌─────────┐
    │  Shiro  ││Redis ││   Kafka    ││  MySQL   ││   ES    │
    │  认证   ││ 缓存 ││  消息队列  ││ 主从复制 ││商品搜索 │
    │  授权   ││Lua脚本││ 削峰填谷  ││ 读写分离 ││全文检索 │
    └─────────┘└──────┘└────────────┘└────┬─────┘└─────────┘
                                          │
                                  ┌───────┴────────┐
                                  │ ShardingSphere  │
                                  │  订单表分库分表  │
                                  └────────────────┘
```
