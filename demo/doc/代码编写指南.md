## 编写步骤（5 步走）

### 第 1 步：设计数据库表

参考 `src/main/resources/sql/schema.sql`：

```sql
-- 示例：用户表
CREATE TABLE IF NOT EXISTS t_user (
    id BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '用户ID',
    username VARCHAR(100) NOT NULL UNIQUE COMMENT '用户名',
    password_hash VARCHAR(256) NOT NULL COMMENT '密码哈希',
    phone VARCHAR(20) COMMENT '手机号',
    status TINYINT DEFAULT 1 COMMENT '状态: 0禁用 1正常',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    UNIQUE INDEX uk_username(username),
    INDEX idx_phone(phone)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
```

**设计要点**：
- 主键使用 `BIGINT AUTO_INCREMENT`
- 添加必要的索引（唯一索引、普通索引）
- 添加 `created_at` 和 `updated_at` 字段
- 注释要详细说明字段含义
- 枚举值要在注释中说明（如：0禁用 1正常）

### 第 2 步：创建实体类（Entity）

路径：`src/main/java/com/example/demo/model/entity/`

```java
package com.example.demo.model.entity;

import lombok.Data;
import java.time.LocalDateTime;

/**
 * 用户实体类
 * 对应数据库表 t_user
 */
@Data
public class User {
    private Long id;
    private String username;
    private String passwordHash;  // 数据库字段 password_hash
    private String phone;
    private Integer status;
    private LocalDateTime createdAt;  // 数据库字段 created_at
    private LocalDateTime updatedAt;  // 数据库字段 updated_at
}
```

**命名规则**：
- 类名：大驼峰（UserEntity → User）
- 属性名：小驼峰（password_hash → passwordHash）
- MyBatis 会自动转换下划线命名

### 第 3 步：创建 DTO 和 VO

#### DTO（Data Transfer Object）- 请求参数

路径：`src/main/java/com/example/demo/model/dto/`

```java
package com.example.demo.model.dto;

import lombok.Data;
import javax.validation.constraints.NotBlank;
import javax.validation.constraints.Size;

/**
 * 用户注册请求 DTO
 */
@Data
public class UserRegisterRequest {

    @NotBlank(message = "用户名不能为空")
    @Size(min = 3, max = 20, message = "用户名长度必须在3-20个字符之间")
    private String username;

    @NotBlank(message = "密码不能为空")
    @Size(min = 6, max = 20, message = "密码长度必须在6-20个字符之间")
    private String password;

    private String phone;
}
```

#### VO（View Object）- 返回数据

路径：`src/main/java/com/example/demo/model/vo/`

```java
package com.example.demo.model.vo;

import lombok.Data;
import java.time.LocalDateTime;

/**
 * 用户信息视图对象 VO
 * 返回给前端，不包含敏感信息（如密码）
 */
@Data
public class UserVO {
    private Long id;
    private String username;
    private String phone;
    private Integer status;
    private LocalDateTime createdAt;
}
```

**DTO vs VO 区别**：
- **DTO**：用于接收前端传来的数据（如：注册、登录、更新）
- **VO**：用于返回给前端的数据（隐藏敏感字段，如密码）

### 第 4 步：创建 Mapper 接口和 XML

#### Mapper 接口

路径：`src/main/java/com/example/demo/mapper/`

```java
package com.example.demo.mapper;

import com.example.demo.model.entity.User;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Param;
import java.util.List;

/**
 * 用户 Mapper 接口
 */
@Mapper
public interface UserMapper {

    // 查询
    User selectById(@Param("id") Long id);
    User selectByUsername(@Param("username") String username);
    List<User> selectList(@Param("offset") Integer offset, @Param("limit") Integer limit);

    // 插入
    int insert(User user);

    // 更新
    int update(User user);
    int updateStatus(@Param("id") Long id, @Param("status") Integer status);

    // 删除
    int deleteById(@Param("id") Long id);
}
```

#### MyBatis XML 映射文件

路径：`src/main/resources/mapper/UserMapper.xml`

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">

<mapper namespace="com.example.demo.mapper.UserMapper">

    <!-- 结果映射 -->
    <resultMap id="BaseResultMap" type="com.example.demo.model.entity.User">
        <id column="id" property="id"/>
        <result column="username" property="username"/>
        <result column="password_hash" property="passwordHash"/>
        <result column="phone" property="phone"/>
        <result column="status" property="status"/>
        <result column="created_at" property="createdAt"/>
        <result column="updated_at" property="updatedAt"/>
    </resultMap>

    <!-- 基础字段列表 -->
    <sql id="Base_Column_List">
        id, username, password_hash, phone, status, created_at, updated_at
    </sql>

    <!-- 根据ID查询 -->
    <select id="selectById" resultMap="BaseResultMap">
        SELECT <include refid="Base_Column_List"/>
        FROM t_user
        WHERE id = #{id}
    </select>

    <!-- 插入用户 -->
    <insert id="insert" useGeneratedKeys="true" keyProperty="id">
        INSERT INTO t_user (username, password_hash, phone, status)
        VALUES (#{username}, #{passwordHash}, #{phone}, #{status})
    </insert>

    <!-- 更新用户 -->
    <update id="update">
        UPDATE t_user
        <set>
            <if test="phone != null">phone = #{phone},</if>
            <if test="status != null">status = #{status},</if>
        </set>
        WHERE id = #{id}
    </update>

    <!-- 删除用户 -->
    <delete id="deleteById">
        DELETE FROM t_user WHERE id = #{id}
    </delete>

</mapper>
```

**XML 配置要点**：
- `namespace` 必须是 Mapper 接口的全限定名
- `<resultMap>` 用于映射数据库字段到实体类属性
- `<sql>` 定义可重用的 SQL 片段
- 使用 `<if>` 标签实现动态 SQL

### 第 5 步：创建 Service 业务逻辑层

路径：`src/main/java/com/example/demo/service/`

```java
package com.example.demo.service;

import com.example.demo.common.ErrorCode;
import com.example.demo.common.Result;
import com.example.demo.mapper.UserMapper;
import com.example.demo.model.dto.UserRegisterRequest;
import com.example.demo.model.entity.User;
import com.example.demo.model.vo.UserVO;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.BeanUtils;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

/**
 * 用户服务实现类
 */
@Slf4j
@Service
public class UserService {

    @Autowired
    private UserMapper userMapper;

    /**
     * 用户注册
     */
    @Transactional(rollbackFor = Exception.class)
    public Result<Long> register(UserRegisterRequest request) {
        // 1. 校验用户名是否已存在
        User existUser = userMapper.selectByUsername(request.getUsername());
        if (existUser != null) {
            return Result.error(ErrorCode.USER_ALREADY_EXISTS);
        }

        // 2. 创建用户对象
        User user = new User();
        user.setUsername(request.getUsername());
        user.setPasswordHash(encryptPassword(request.getPassword()));
        user.setPhone(request.getPhone());
        user.setStatus(1);

        // 3. 插入数据库
        int rows = userMapper.insert(user);
        if (rows > 0) {
            log.info("用户注册成功，userId={}", user.getId());
            return Result.success(user.getId());
        }

        return Result.error(ErrorCode.SYSTEM_ERROR);
    }

    /**
     * 获取用户信息
     */
    public Result<UserVO> getUserById(Long userId) {
        User user = userMapper.selectById(userId);
        if (user == null) {
            return Result.error(ErrorCode.USER_NOT_FOUND);
        }

        UserVO userVO = new UserVO();
        BeanUtils.copyProperties(user, userVO);
        return Result.success(userVO);
    }

    /**
     * 密码加密（示例）
     */
    private String encryptPassword(String password) {
        // TODO: 使用 BCrypt 加密
        return "hashed_" + password;
    }
}
```

**Service 层要点**：
- 使用 `@Service` 注解标记
- 使用 `@Transactional` 保证事务一致性
- 使用 `@Slf4j` 记录日志
- 统一返回 `Result<T>` 类型
- 错误情况返回对应的错误码

### 第 6 步：创建 Controller 控制器层

路径：`src/main/java/com/example/demo/controller/`

```java
package com.example.demo.controller;

import com.example.demo.common.Result;
import com.example.demo.model.dto.UserRegisterRequest;
import com.example.demo.model.vo.UserVO;
import com.example.demo.service.UserService;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.validation.annotation.Validated;
import org.springframework.web.bind.annotation.*;

/**
 * 用户 Controller
 */
@Slf4j
@RestController
@RequestMapping("/api/user")
public class UserController {

    @Autowired
    private UserService userService;

    /**
     * 用户注册
     * POST /api/user/register
     */
    @PostMapping("/register")
    public Result<Long> register(@Validated @RequestBody UserRegisterRequest request) {
        log.info("用户注册请求，username={}", request.getUsername());
        return userService.register(request);
    }

    /**
     * 获取用户信息
     * GET /api/user/{id}
     */
    @GetMapping("/{id}")
    public Result<UserVO> getUserById(@PathVariable Long id) {
        log.info("查询用户信息，userId={}", id);
        return userService.getUserById(id);
    }

    /**
     * 更新用户信息
     * PUT /api/user/{id}
     */
    @PutMapping("/{id}")
    public Result<Void> updateUser(@PathVariable Long id,
                                     @RequestBody UserUpdateRequest request) {
        return userService.updateUser(id, request);
    }

    /**
     * 删除用户
     * DELETE /api/user/{id}
     */
    @DeleteMapping("/{id}")
    public Result<Void> deleteUser(@PathVariable Long id) {
        return userService.deleteUser(id);
    }
}
```

**Controller 层要点**：
- 使用 `@RestController` 注解
- 使用 `@RequestMapping` 定义基础路径
- RESTful API 设计：
  - GET：查询
  - POST：创建
  - PUT：更新
  - DELETE：删除
- 使用 `@Validated` 验证请求参数
- 记录请求日志

## RESTful API 设计规范

### URL 命名规范

```
GET    /api/user/{id}        # 获取用户信息
POST   /api/user/register    # 用户注册
POST   /api/user/login       # 用户登录
PUT    /api/user/{id}        # 更新用户信息
DELETE /api/user/{id}        # 删除用户
```

### HTTP 状态码

| 状态码 | 说明 | 使用场景 |
|-------|------|---------|
| 200 | OK | 请求成功 |
| 400 | Bad Request | 参数错误 |
| 401 | Unauthorized | 未登录 |
| 403 | Forbidden | 无权限 |
| 404 | Not Found | 资源不存在 |
| 500 | Internal Server Error | 系统错误 |

```json

    "message": "success",
    "data": {

        "username": "testuser"
```

## MyBatis 常用操作

### 动态 SQL

```xml
<!-- 条件查询 -->
<select id="selectByCondition" resultMap="BaseResultMap">
    SELECT * FROM t_user
    <where>
        <if test="username != null">
            AND username LIKE CONCAT('%', #{username}, '%')
        </if>
        <if test="status != null">
            AND status = #{status}
        </if>
    </where>
</select>

<!-- 批量插入 -->
<insert id="batchInsert">
    INSERT INTO t_user (username, phone) VALUES
    <foreach collection="list" item="user" separator=",">
        (#{user.username}, #{user.phone})
    </foreach>
</insert>

<!-- 批量更新 -->
<update id="batchUpdate">
    <foreach collection="list" item="user" separator=";">
        UPDATE t_user SET status = #{user.status} WHERE id = #{user.id}
    </foreach>
</update>
```

### 分页查询

```xml
<!-- 方式1：LIMIT 分页 -->
<select id="selectList" resultMap="BaseResultMap">
    SELECT * FROM t_user
    ORDER BY created_at DESC
    LIMIT #{offset}, #{limit}
</select>

<!-- 方式2：使用 PageHelper 插件（推荐） -->
<!-- 在 Service 层使用：PageHelper.startPage(page, size); -->
```

## 常见问题

### 1. 下划线字段名转驼峰

**方式1**：在 `application.yml` 配置

```yaml
mybatis:
  configuration:
    map-underscore-to-camel-case: true
```

**方式2**：在 XML 中手动映射

```xml
<resultMap id="BaseResultMap" type="User">
    <result column="password_hash" property="passwordHash"/>
    <result column="created_at" property="createdAt"/>
</resultMap>
```

### 2. 插入后获取自增主键

```java
@Insert("INSERT INTO t_user (...) VALUES (...)")
@Options(useGeneratedKeys = true, keyProperty = "id")
int insert(User user);

// 或在 XML 中
<insert id="insert" useGeneratedKeys="true" keyProperty="id">
    ...
</insert>
```

### 3. 事务管理

```java
@Transactional(rollbackFor = Exception.class)
public void someMethod() {
    // 所有 Mapper 操作在同一事务中
    // 任何异常都会回滚
}
```

## 总结

**5 步编写一个完整模块**：

1. ✅ **设计数据库表** → `schema.sql`
2. ✅ **创建实体类** → `model/entity/`
3. ✅ **创建 DTO/VO** → `model/dto/` 和 `model/vo/`
4. ✅ **创建 Mapper** → `mapper/` + `resources/mapper/`
5. ✅ **创建 Service** → `service/`
6. ✅ **创建 Controller** → `controller/`

**参考示例**：用户模块（User）已完整实现，可直接复制并修改为其他模块。
