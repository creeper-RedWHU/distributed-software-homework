## 代码特点

### 1. 完整的分层架构

```
Controller (REST API 层)
    ↓
Service (业务逻辑层)
    ↓
Mapper (数据访问层)
    ↓
Database (MySQL)
```

### 2. MyBatis 最佳实践

✅ 使用 XML 方式配置 SQL（便于维护复杂查询）
✅ 使用 `<resultMap>` 映射字段
✅ 使用 `<sql>` 定义可复用片段
✅ 动态 SQL 支持（`<if>`、`<where>`）
✅ 自动获取自增主键

### 3. RESTful API 规范

✅ GET - 查询
✅ POST - 创建
✅ PUT - 更新
✅ DELETE - 删除
✅ 统一响应格式（Result<T>）
✅ 参数验证（@Validated）

### 4. 安全设计

✅ 密码加密（预留 BCrypt 接口）
✅ Token 认证（预留 JWT 接口）
✅ VO 隐藏敏感信息
✅ 参数校验防止非法输入

### 5. 事务管理

✅ 使用 `@Transactional` 保证数据一致性
✅ 异常回滚机制
