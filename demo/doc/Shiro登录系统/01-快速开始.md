## ⚙️ 核心配置

### 版本信息

⚠️ **重要**: 由于 Shiro 2.0.1 不支持 Jakarta EE，项目使用 **Spring Boot 2.7.18**

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 2.7.18 | 使用 `javax.servlet`（兼容 Shiro）|
| Apache Shiro | 2.0.1 | 核心库（shiro-spring + shiro-web）|
| Java | 17 | 支持 Java 8+ |

**为什么不用 Spring Boot 4.0.3?**
Spring Boot 4.x 使用 Jakarta EE（`jakarta.servlet.*`），而 Shiro 2.0.1 仍使用 Java EE（`javax.servlet.*`），不兼容。

**详细说明**: [SPRING_BOOT_DOWNGRADE.md](SPRING_BOOT_DOWNGRADE.md)

### 主要特性
- ✅ 基于 Apache Shiro 的认证和授权
- ✅ 双表设计（用户信息表 + 用户登录表）
- ✅ 三种用户类型（商家/买家/管理员）
- ✅ SHA-256 + 盐值密码加密
- ✅ 细粒度权限控制
- ✅ Session 缓存支持

## 🎯 主要 API

```bash
# 注册
POST /api/auth/register/merchant  # 商家注册
POST /api/auth/register/buyer     # 买家注册

# 登录/登出
POST /api/auth/login              # 登录
POST /api/auth/logout             # 登出

# 用户信息
GET  /api/auth/info               # 获取当前用户信息
POST /api/auth/change-password    # 修改密码

# 权限测试
GET  /api/auth/test/merchant      # 测试商家权限
GET  /api/auth/test/buyer         # 测试买家权限
GET  /api/auth/test/admin         # 测试管理员权限
```
